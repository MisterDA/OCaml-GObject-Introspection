Module Loader

module Loader: sig .. end
Loader : type and functions in order to read the GObject-Introspection of an instrospectable library and generate Ctypes bindings.
type t
a Loader.t is a GObject-Instrospection namespace that has been loaded.
val load : string -> ?version:string -> unit -> t option
Load a namespace with an optional version.
val get_namespace : t -> string
Get the namespace of the loader.
val get_version : t -> string
Get the version of the loaded namespace.
val generate_dir : t -> unit
val set_build_path : t -> string -> t
Specify the root path where the directories and the files will be created. For example, with path = "toto/titi/tata" , with a namespace of "GLib", then the files will go in "toto/titi/tata/GLib/lib"
val get_build_path : t -> string
Get the build path of the loader
val get_lib_path : t -> string
Get the path where the files will be build : build_path/namespace/lib
val generate_main_module_files : t -> Builder.files
Generate the files Namespace.ml and Namespace.mli in append mode
val generate_directories : t -> unit
Generate the directories build_path/namespace/lib
val parse : t ->
       ?struct_parser:(GIBaseInfo.t Ctypes.structure Ctypes.ptr ->
                       Builder.files -> unit) ->
       unit -> unit
Parse the GI infos and create the corresponding ctypes code
struct_parser : allow user to customize the GIStructureInfo bindings generator
