| Not_implemented of string

val type_tag_to_bindings_types : GITypes.tag ‑> bindings_types

Obtain from a GITypes.tag the type strings to use in bindings for a GITypes.tag if implemented. Returns Not_implemented with the tag name if not implemented. This is for simple scalar type (ie. with BuilderEnum.get_storage_type) .

val type_info_to_bindings_types : GITypeInfo.t Ctypes.structure Ctypes.ptr ‑> bindings_types

Obtain from a GITypeInfo.t the type strings to use in bindings. Returns Not_implemented with the tag name if not implemented.
