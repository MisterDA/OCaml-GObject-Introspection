val type_tag_to_ctypes_strings : GITypes.tag -> string * string
Transforms a GITypes.tag to the corresponding types for the mli file and the ml file. For example, the GITypes.Int8 tag returns ("int", "int8_t"). The "int" string is used in the OCaml header file for signature while the int8_t is used in OCaml file for Ctypes types usage.
val type_tag_to_ctypes_typ_string : GITypes.tag -> string
Get the Ctypes typ representation of a GITypes.tag. For example, if the tag is GITypes.Int8, it will returns "int8_t".
