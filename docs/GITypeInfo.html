GITypeInfo — Struct representing a type
type t
GITypeInfo represents a type. You can retrieve a type info from an argument (see GIArgInfo), a function return value (see GIFunctionInfo), a field (see GIFieldInfo), a property (see GIPropertyInfo), a constant (see GIConstantInfo) or for a union discriminator (see GIUnionInfo). A type can either be a of a basic type which is a standard C primitive type or an interface type. For interface types you need to call GITypeInfo.get_interface to get a reference to the base info for that interface.

val get_interface : t Ctypes.structure Ctypes.ptr ->
       GIBaseInfo.t Ctypes.structure Ctypes.ptr option
For types which have GITypes.Interface such as GObjects and boxed values, this function returns full information about the referenced type. You can then inspect the type of the returned GIBaseInfo to further query whether it is a concrete GObject, a GInterface, a structure, etc. using GIBaseInfo.get_type.
