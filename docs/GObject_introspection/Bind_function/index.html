Bind_function : regroups all functions needed to parse and generate the OCaml code for functions.

type argument =

pre : string;

| Arg_in_out of {

Each argument of a function can be represented with this type based on the direction of the argument. The Arg_in form is the simpler. The two other forms need to provide in the ml file some pre code and post code for the implementation of the ctypes bindings.

type arguments =

| List of argument list

The set of arguments of a function. The full set of the arguments must be parsed before knowing if we can implement the function hence the Not_implemented variant. TODO : remove / replace with func_args.
